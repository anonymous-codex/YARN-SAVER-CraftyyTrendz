# 🧶 Yarn-Saver

**Yarn-Saver** is a digital utility designed for fiber artists who need to know exactly how far their yarn will take them. No more guessing, no more frog-ing—just precise, data-driven crafting.

Built by **Crafty Trendz**.

## ✨ Features
* **Precision Row Calculation:** Uses your specific swatch weight and project width to calculate remaining rows.
* **Safety Buffer Mode:** Automatically subtracts a 2-row "cushion" to ensure you never run out of yarn mid-row.
* **Modern Aesthetic:** A clean, "Sage & Cream" glassmorphism interface with a tactile polka-dot workspace.
* **Input Protection:** Strict numeric-only fields to prevent calculation errors.
* **Mobile Optimized:** Designed to be used right at your crafting desk or on the go.

## 🛠️ How It Works
The app calculates the **Grams Per Row** using the following formula:
$GramsPerRow = (SwatchWeight / 100) \times ProjectWidth$

It then divides your **Remaining Yarn** by that value to give you an estimated row count.

## 🚀 Quick Start
1. **Weigh your swatch** (e.g., a 10x10cm square).
2. **Measure your project's width** in centimeters.
3. **Weigh your remaining yarn** ball.
4. Input the values and hit **Calculate Rows**.
